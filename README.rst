
==========
Cloud-sync
==========

Environment variables
=====================

``VMNAME``
    Virtual machine name

``TARGET``
    Name of directory to sync

``SOURCEBASE``
    Path to root of sync content, in local filesystem

``DESTBASE``
    Path to root of sync destination, in remote SFTP filesystem

``DESTUSER``
    SFTP username

``DESTHOST``
    SFTP server

``DESTPORT``
    SFTP port
